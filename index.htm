How to git.

Step by step git

  1. mkdir and files
  2. cd into that dir
  3. git init
  4. git add .
  5. git status
  6. git commit -m 'initial commit'
  7. git branch start-files
  8. branch finished-files
  9. git checkout finished-files
  10. echo "# three-col-exercise" >> README.md
    git add README.md
